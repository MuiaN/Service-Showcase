# Service Showcase Website

A premium, responsive portfolio and service showcase website built with React, Vite, and Tailwind CSS. This project features smooth animations, a modern glassmorphism aesthetic, and a comprehensive pricing and service display for creative digital services.

## Features

- 🎨 **Modern Design**: Styled with Tailwind CSS v4 featuring a dark/glass theme.
- ⚡ **High Performance**: Built on Vite for fast development and bundling.
- 🎭 **Animations**: Interactive UI elements powered by Framer Motion.
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop views.
- 🧩 **Component Architecture**: Modular UI using Radix UI primitives.
- 🛣️ **Routing**: Lightweight client-side routing with wouter.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, clsx, tailwind-merge
- **Animations**: Framer Motion
- **UI Components**: Radix UI, Lucide React (Icons)
- **Routing**: wouter
- **State Management**: React Query (TanStack Query)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`

## Scripts

- `npm run dev`: Start the development server
- `npm run build`: Build the application for production
- `npm run serve`: Preview the production build locally
- `npm run typecheck`: Run TypeScript type checking
